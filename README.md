# Requirement

Docker

# Run Docker

`
docker-compose up
`

# Access

[Online Website](http://localhost:8000/)

[PhP MyAdmin](http://localhost:8080/)

# Database

It's STRONGLY advised to change these in the docker-compose.yml

** Host **
> db

** Database name **
> MYSQL_DATABASE

## Root Access

** Username **
> root

** Password **
> MYSQL_ROOT_PASSWORD

## User Access

** Username **
> MYSQL_USER

** Password **
> MYSQL_PASSWORD
